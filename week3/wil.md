+ CSS의 기본 margin과 padding 공부하기
margin과 padding 모두 여백을 만드는 속성.
1. margin : 요소 바깥쪽 여백 생성(다른 요소와의 간격을 조절)
배경색 영향 없음
ex) ```.box { margin: 20px; }```
= 모든 방향에 20px 바깥 여백
- 방향별로도 가능
```margin-top: 10px;```
```margin-right: 20px;```
```margin-bottom: 10px;```
```margin-left: 20px;```
- 한 번에 모든 방향 컨트롤 가능
```margin: 10px 20px 30px 40px;```
top -> right -> bottom -> left (시계 방향)

2. padding : 요소 안쪽 여백 생성(콘텐츠와 테두리 사이 내부 공간 조절)
배경색 영향 있음(색이 채워짐) 
ex) ```.box { padding: 20px; }```
= 안쪽이 20px 넓어짐(박스 자체가 커짐)
