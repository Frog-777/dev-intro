- git clone <repository 링크> : 만들어놨던 repository 복제됨
- ls : 이 위치에 어떤 폴더가 있는지 읽어줌
- cd + 폴더 이름 -> 폴더 안으로 들어가짐
- cd .. -> 폴더 밖으로 나가짐
- git add. -> 파일 한 번에 올리기
